## 1. Разминка: вспоминаем всё

Напиши и запусти эту программу. Она просит имя и возраст, а затем говорит, сколько тебе будет через 5 лет.

```python
имя = input("Привет! Как тебя зовут? ")
возраст = input("Сколько тебе лет? ")
возраст = int(возраст)   # превращаем строку в число
будущее = возраст + 5
print("Рад познакомиться, " + имя + "!")
print("Через 5 лет тебе будет", будущее, "лет.")
```

Что мы повторили:

- `input(...)` всегда выдаёт строку (текст).
- Чтобы складывать числа, строку нужно превратить в целое число с помощью `int()`.
- `print` может выводить несколько штук через запятую.
- Если что-то забыл – не страшно, дальше всё пригодится.

## 2. Жизнь — это выбор

Представь: ты выходишь на улицу. Если идёт дождь, ты берёшь зонт, иначе идёшь без него.

Схема выглядит так:

```text
ЕСЛИ дождь?
 ├── да → взять зонт
 └── нет → не брать зонт
```

Компьютер умеет делать точно так же, только вопросы он задаёт по-своему.  
В Python для этого есть волшебные слова:

- `if` («если»)
- `else` («иначе»)

## 3. Первый условный оператор

Давай напишем программу, которая проверяет возраст для входа в кинотеатр. Если тебе 18 или больше — пускают, иначе — нет.

```python
возраст = int(input("Сколько тебе лет? "))

if возраст >= 18:
    print("Проходи, вот твой билет!")
else:
    print("Извини, этот фильм только для взрослых.")
```

Запусти программу несколько раз с разными числами. Обрати внимание:

- Когда вводишь 20 — срабатывает первая строчка внутри `if`.
- Когда вводишь 10 — срабатывает та, что внутри `else`.

Как это работает:

- `>=` означает «больше или равно». Другие знаки сравнения:
  - `>` — больше
  - `<` — меньше
  - `==` — равно (два знака, потому что один `=` уже занят под присваивание)
  - `!=` — не равно
- После `if условие:` обязательно ставится двоеточие `:` и следующая строчка пишется с отступом (4 пробела или Tab). Всё, что с таким отступом, принадлежит этому `if`.
- `else:` тоже с двоеточием, и его блок тоже с отступом.
- Отступы — это способ сказать компьютеру: «Вот эти команды выполняй только при таком условии». Без них ничего не заработает.

## 4. Вложенные условия

А если правил больше? Например:

- Дети до 12 лет получают скидку.
- Подростки от 12 до 18 — только с родителями.
- Взрослые от 18 проходят свободно.

Можно вложить одно `if-else` внутрь другого:

```python
возраст = int(input("Сколько тебе лет? "))

if возраст >= 18:
    print("Полный билет, проходи!")
else:
    print("Ты ещё не взрослый.")
    if возраст < 12:
        print("Для детей до 12 — билет со скидкой 50%!")
    else:
        print("С 12 до 18 можно только с родителями.")
```

Посмотри на «лесенку»:

- Внешний `else` выполняется, когда возраст меньше 18.
- Внутри него сидит свой собственный `if-else`, который проверяет, меньше ли 12.
- Таким способом можно строить целые деревья решений.

## 5. Несколько условий подряд — `elif`

Когда выбор из нескольких вариантов, удобнее использовать конструкцию `if – elif – else`. `elif` — сокращение от «else if» (иначе если). Это как новые ступеньки проверки:

```python
возраст = int(input("Сколько тебе лет? "))

if возраст < 12:
    print("Детский билет со скидкой.")
elif возраст < 18:
    print("Подростковый билет, нужен взрослый.")
else:
    print("Взрослый билет.")
```

Компьютер проверяет условия по очереди и выполняет только первый подошедший блок. Поэтому самое узкое условие (`<12`) пишем первым, иначе оно никогда не выполнится.

## 6. Собираем всё вместе: игра «Угадай число»

Теперь применим условия для настоящей игры. Компьютер загадает случайное число от 1 до 5, а ты попробуешь угадать с одной попытки.

```python
import random  # ящик с инструментами для случайностей

секрет = random.randint(1, 5)  # компьютер загадывает число от 1 до 5
print("Я загадал число от 1 до 5. Попробуй угадать!")
догадка = int(input("Твой вариант: "))

if догадка == секрет:
    print("Потрясающе! Ты угадал! Это действительно", секрет)
else:
    print("Не угадал. Я загадал", секрет)
    if догадка > секрет:
        print("Твоё число слишком большое.")
    else:
        print("Твоё число слишком маленькое.")
```

Разбор новых моментов:

- `import random` — мы подключили модуль `random`, в котором лежат готовые функции для случайных чисел. Представь, что это ящик с инструментами, мы просто берём то, что нужно.
- `random.randint(1, 5)` — специальная функция, которая выдаёт случайное целое число от 1 до 5 (включая границы).
- Секретное число теперь меняется каждый раз, поэтому игра становится непредсказуемой.
- Внутри `else` (когда не угадал) мы даём подсказку «больше» или «меньше», чтобы игроку было легче в следующей попытке.

Запусти игру несколько раз, проверь, что подсказка работает верно.
