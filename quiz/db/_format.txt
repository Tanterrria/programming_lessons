Формат файла квиза (JSON, UTF-8)
================================

Один файл = один квиз. Поля:

  id            — строка, латиница/дефис, уникально среди файлов
  title         — заголовок для списка
  description   — опционально, подзаголовок
  questions     — массив вопросов

Каждый вопрос:

  text          — текст вопроса (можно многострочный, допустимы переносы \n)
  options       — ровно 4 строки, варианты a–d
  correctIndex  — целое 0..3 (индекс верного варианта)
  comment       — опционально: пояснение после выбора ответа (в viewer/builder)

Добавить квиз в просмотр:
  1) Положите файл .json в эту папку (quiz/db/).
  2) Добавьте запись в manifest.json в массив "quizzes" (id, title, file).

Конструктор (quiz/builder/) экспортирует JSON в этом формате — сохраните файл
в quiz/db/ и пропишите его в manifest.json.

Запуск: из корня репозитория
  python3 -m http.server 8765

Просмотр:  http://127.0.0.1:8765/quiz/viewer/
Конструктор: http://127.0.0.1:8765/quiz/builder/
